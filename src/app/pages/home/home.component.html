Este projeto é um showcase completo desenvolvido em Angular 19 com Angular CLI, integrando o Design System Gov.br via @govbr-ds/webcomponents-angular (v2.0.0-next.37).

Inspirado em padrões modernos e no Design System Gov.br, este projeto oferece uma referência prática e elegante para desenvolvedores.

Com foco em integração, customização e reutilização, aceleramos a adoção de componentes em aplicações Angular.
